test 插件的作用是执行项目中的单元测试,你可以根据需要修改自定义脚本。

### PHP

通过 PHPUnit 执行单元测试:

.docs-content article .inner pre {
    border: none;
    box-shadow: 0 0 2px 2px rgba(0,0,0,.1) inset;
    background: #313639;
    color: #f9e890;
}<code> phpunit </code>

### Node.js

执行 npm test 命令，通过 NPM 命令执行单元测试:

> npm test

### Ruby

通过 Bundle 命令执行单元测试:

> bundle exec rake 

### Python

通过 NPM 命令执行单元测试：

> make test
