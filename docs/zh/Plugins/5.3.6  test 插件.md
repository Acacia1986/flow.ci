test 插件的作用是执行项目中的单元测试,你可以根据需要修改自定义脚本。

### PHP

通过 PHPUnit 执行单元测试:

<pre>
<code> phpunit </code>
</pre>


### Node.js

执行 npm test 命令，通过 NPM 命令执行单元测试:

<pre>
<code> npm test </code>
</pre>

### Ruby

通过 Bundle 命令执行单元测试:

<pre>
<code> bundle exec rake </code>
</pre>
 

### Python

通过 NPM 命令执行单元测试：

<pre>
<code> make test </code>
</pre>
