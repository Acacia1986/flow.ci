# Ruby

Flow.ci 当前支持以下 Ruby 版本：

- Ruby 1.9.3
- Ruby 2.0.0
- Ruby 2.1.0
- Ruby 2.1.2
- Ruby 2.2.0
- Ruby 2.2.2

## 依赖安装
以下的 Ruby 依赖管理工具已经默认安装支持：

- bundler
- rubygems

Ruby 项目默认使用 Bundler 作为依赖管理工具：
